Markdown Cheat Sheet

# H1, ## H2, ### H3 — Headings
**bold** — Bold text
*italic* — Italic text
`code` — Inline code
- item — Unordered list
1. item — Ordered list
[text](url) — Link
![alt](url) — Image
> text — Blockquote
--- — Horizontal rule
