What Is a GLN?

A Global Location Number is a 13-digit GS1 identifier used to represent legal entities, functional entities, or physical locations. It is built from a GS1 Company Prefix, a location reference, and a GS1 check digit.

What Is an SGLN?

An SGLN extends the location identity by adding a serial or extension component. That makes it useful when you need to identify a more specific location instance or business context.

How Generation Works

At SmartToolsHub, GCP generation uses the prefix and length rules from the GS1 reference file. The location reference fills the remaining base digits needed to form a 12-digit GLN body, and the final check digit is calculated using the GS1 Mod 10 method.

Important Reminder

A structurally valid GLN or SGLN is not the same as an officially licensed identifier. Real-world verification still depends on the underlying GS1 Company Prefix and registration details.
