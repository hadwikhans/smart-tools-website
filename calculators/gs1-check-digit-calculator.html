What is a check digit?

The last digit of all fixed-length numeric GS1 identification keys is a check digit that ensures integrity of the key. Most GS1 keys need a check digit, except some specific identifiers with different structures.

How does the check digit calculator work?

The check digit is calculated using the GS1 Mod 10 algorithm, based on the other numbers in the key.
